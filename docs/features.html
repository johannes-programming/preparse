The PreParser class

This class is the centerpiece of the preparse.

class preparse.PreParser(optdict: Any = None, prog: Any = None, abbrev: Any = preparse.Abbrev.COMPLETE, permutate: Any = True, posix: Any = "infer")

Create a new PreParser object.

abbrev

This property determines the approach of the instance to abbreviations of long options.

click(parser: Any, cmd: Any = True, ctx: Any = True) -> None

This method returns a decorator that infuses the current instance into parse_args.

clickCommand(cmd: click.Command) -> None

This method alters the current instance to reflect a click.Command object.

clickContext(ctx: click.Context) -> None

This method alters the current instance to reflect a click.Context object.

copy() -> Self

This method returns a copy of the current instance.

optdict

This property holds a dictionary. Its keys are the options. Its values specify nargs.

parse_args

This method is at the core of the class. It returns a preparsed list of arguments.

permutate

This property decides whether the arguments are permutated in parsing.

posix

This property decides if posix parsing is used, i.e. a positional argument causes all the arguments after it to be also interpreted as positional. If the value passed to the property equals "infer" then os.environ.get("POSIXLY_CORRECT") is saved.

prog

This property represents the name of the program. If the value passed to the property is None then str(sys.argv[0]) is saved.

todict() -> dict

This method a dict representing the current instance.

warn(message: Any) -> None

This method lauches a warning about something.

warnAboutUnrecognizedOption(option: Any) -> None

This method lauches a warning about an unrecognized option.

warnAboutInvalidOption(option: Any) -> None

This method lauches a warning about an invalid option.

warnAboutAmbiguousOption(option: Any, possibilities: Iterable) -> None

This method lauches a warning about an ambiguous option.

warnAboutUnallowedArgument(option: Any) -> None

This method lauches a warning about an unallowed argument.

warnAboutRequiredArgument(option: Any) -> None

This method lauches a warning about a required argument.
